1.Hardware Architecture — Overview

Every computer — desktop, laptop, or smartphone — is built from the same fundamental building blocks. Hardware components fall into four categories: Processor, Memory, Input/Output Devices, and Storage.

Section 02

2.The Four Hardware Components

🧠
Processor (CPU)
The brain of the computer. It fetches, decodes, and executes instructions from software and the user. Modern CPUs contain multiple cores allowing simultaneous task execution.
💾
Memory (RAM & ROM)
Fast, temporary workspace for the CPU. RAM holds programs and data currently in use; ROM stores permanent boot instructions. More RAM means a faster, more capable system.
⌨️
Input & Output Devices
Devices that let users interact with the computer. Input: keyboard, mouse, scanner, microphone Output: monitor, printer, speakers. Some devices (touchscreen) do both.
🗄️
Storage (Secondary Memory)
Persistent storage that retains data when powered off. Much slower than RAM but far larger capacity. Examples: HDD, SSD, USB flash drives.
Section 03

3.The Processor (CPU) in Detail

A CPU is a microchip etched with millions of tiny circuits on silicon. It contains three essential sub-components:

🎛️
Control Unit (CU)
It acts as a traffic manager — directs the flow of data between the CPU, memory, and other components. It does not process data itself; it coordinates everything.
➕
Arithmetic Logic Unit (ALU)
It does the actual computation — addition, subtraction, multiplication, division, and logical comparisons (equal to, greater than, less than, not equal to, etc.). All computer data is numbers; the ALU crunches them.
📋
Registers
Ultra-fast, tiny memory cells inside the CPU that store the data being processed right now. Think of them as a scratchpad — data is loaded here before an arithmetic or logical operation.
📱
Smartphone Perspective — The Processor
Your smartphone has a CPU too — it is called a SoC (System on a Chip). Everything — CPU, GPU, RAM controller — is packed onto one tiny chip. When you open a heavy game and your phone gets warm, that is your SoC working hard and generating heat, just like a desktop CPU. Higher-end phones (e.g. those with Snapdragon 8 Gen series or Apple A-series chips) feel better and perform faster because their SoCs have more cores running at higher speeds.

CPU Feature What It Means
Clock Speed (GHz) Every CPU has a clock driven by a quartz crystal, which can vibrate millions of times per second. The computer uses the vibrations to time its processing operations. In theory, the CPU can execute a function on every tick of the system clock (a clock cycle). However, in practice, the CPU is sometimes idle due to a delay between the request to retrieve data from memory and its delivery. A delay caused by waiting for another component to deliver data is called latency. The clock speed is measured in GHz. GHz is the number of operations a CPU can perform per second (in billions). 1.94 GHz = 1.94 billion operations per second.
Multiple Cores Modern CPUs have multiple cores, so they can complete multiple tasks simultaneously. A core consists of a separate set of essential processor components (control unit, ALU, and registers). Most Intel Core i7 (11th generation) processors typically have eight cores, for example. All the CPU’s cores are located on the same chip.
Hyper-Threading (SMT) CPUs now also support hyper-threading which im plements the concept Simultaneous Multithreading (SMT) Where a single core will present itself as multiple logical cores to a computer’s operating system
CPU Cache (L1/L2/L3)

To help minimize latency, CPUs have caches. A cache is a small amount of very fast memory located near (or within) the CPU. Data that the CPU has recently used or is predicted to need soon is placed in the cache for temporary storage. That way, if the CPU requests the data, it’s more readily available and incurs less delay.

Since the late 1980s, most PC CPUs have had cache memory built into them. This CPU-resident cache is often called the level-1 cache, or L1 cache.

To add even more speed an additional cache is added to CPUs known as Level-2 cache, or L2 cache. This cache used to be found on the motherboard but modern CPUs place L2 cache on the CPU to greatly increase CPU response. L2 cache is slower than L1 cache, but bigger in size.

An additional motherboard-resident cache is called Level-3 cache or L3 cache. L3 cache is the largest cache memory unit, and also the slowest one.

L1, L2, and L3 all speed up the CPU, although in different ways. L1 cache holds instructions that the CPU core is currently using or will need immediately. L2 acts as a secondary, larger buffer to store data that couldn’t fit in the L1 cache but is still likely to be needed soon. L3 serves as a “last resort” cache before hitting main memory (RAM). It is designed to hold larger datasets and shared data, minimizing the need to access the slow RAM. In all cases, the cache memory is faster for the CPU to access, resulting in quicker program execution.

Section 04

4.Memory

⚡
RAM — Random Access Memory
RAM is a temporary, fast working memory that holds programs and data that the CPU is actively using currently. RAM is Volatile, which means everything is lost when it is powered off. More RAM means better multitasking.
🔒
ROM — Read-Only Memory
ROM is a permanent memory that stores the computer's startup (boot) instructions. Non-volatile — data survives power loss. Users cannot (normally) write to it.

ROM stores the BIOS (Basic Input/Output System) — the permanent firmware that runs the instant you press the power button, before any operating system loads. It checks that all hardware is present and working (this is called POST — Power-On Self-Test) and then hands control over to the OS. Unlike RAM, the data here is not erased when power is lost.

📌 Analogy: Think of RAM as your desk (fast, limited working space) and storage as your filing cabinet (slow, large, permanent). The CPU can only directly work with what's on the desk (in RAM).
📱
Smartphone Perspective — RAM
People often confuse a phone's Storage with its RAM. If you try to install an app and see a 'Not enough space' message, that is your Storage running out. But if your phone slows down, or apps constantly reload when you switch between them, that is a RAM issue. When you open multiple apps at once — say WhatsApp, YouTube, and Chrome — each one occupies a portion of RAM. When RAM runs out, the phone starts closing background apps to free up space. This is why more RAM = better multitasking. A phone with 4 GB of RAM struggles to keep many apps open simultaneously, whereas a phone with 12 GB handles it smoothly.
Section 05

5.Storage (Secondary Memory)

Storage retains data permanently even without power. It is much slower than RAM but offers far greater capacity at lower cost.

Type Description Speed
HDD (Hard Disk Drive) Mechanical spinning disk. Stores data magnetically on rotating platters. Slower
SSD (Solid State Drive) No moving parts. Stores data in flash memory chips. Far faster than HDD. Much Faster
USB Flash / Memory Card Portable flash memory for transferring files between devices. Moderate
📱
Smartphone Perspective — Storage
When you buy a phone advertised as "128 GB", that is the phone's storage (secondary memory) — not RAM. This is where your photos, videos, apps, and music are saved permanently. It is the equivalent of an SSD in a laptop. When you get the "Storage full" warning and can no longer take new photos, you have run out of this storage — not RAM.
Section 06

6.Input / Output Devices & Network

⌨️
Input Devices
Accept data from the user and send it to the CPU. Examples: keyboard, mouse, microphone, scanner, camera, touchscreen.
🖥️
Output Devices
Deliver processed results from the CPU to the user. Examples: monitor, printer, speakers, projector.
🌐
Network Connection
Allows the computer to communicate with other devices and the internet. Can be thought of as a very slow, remote form of secondary memory — you retrieve data from it, but it is not always reliable.
📱
Smartphone Perspective — I/O & Network
Your smartphone is almost entirely made of I/O devices. The touchscreen is both an input (you tap it) and an output (it shows you content). The camera is an input device; the speaker is an output device. The microphone is input. The Wi-Fi and mobile data (4G/5G) chip is your network connection — without it, your phone cannot reach the internet, just like a desktop without a network card.
Section 07

7.The Motherboard — A Visual Walkthrough

The motherboard is the main circuit board that connects all hardware components together. Every component discussed above either sits on the motherboard or connects to it. The board below is an Esonic micro-ATX motherboard — a real example used in everyday desktop computers.

▸ Esonic Micro-ATX Motherboard (photographed 2026)
1
CPU Socket (LGA)
The large square socket in the lower-centre of the board. The processor (CPU) plugs in here. The metal lever locks it in place.
2
RAM Slots (DIMM)
The long blue/black slots along the left edge. RAM sticks (one is installed here in green) slide into these.
3
Chipset Heatsink (Northbridge)
The silver finned block near the centre-top labelled "Esonic". It cools the chipset chip beneath it (Northbridge) — the controller that manages communication between CPU, RAM, and PCIe slots.
4
PCIe Slot (x16 — Blue)
The long blue slot near the top. Primarily used for the dedicated graphics card (GPU). PCIe (Peripheral Component Interconnect Express) is the fastest expansion slot standard.
5
PCIe / PCI Slots (Smaller)
Shorter expansion slots for adding additional components such as network cards, sound cards, or Wi-Fi adapters.
6
SATA Connectors
The small L-shaped teal/blue ports along the upper-left edge. Data cables from HDDs and SSDs plug here, connecting storage devices to the motherboard.
7
ATX Power Connector (24-pin)
The large white 2×12 pin block on the right side. The main power cable from the PSU (Power Supply Unit) plugs here to power the entire motherboard.
8
CPU Power Connector
The smaller connector near the power connector. Supplies dedicated power directly to the CPU, separate from the main board power.
9
I/O Panel (Back Panel)
The cluster of ports on the right edge — USB ports (rectangular), Ethernet (RJ45), audio jacks, video output(VGA), Parallel Port (LPT), Serial Port (COM) — Male. These are what you see on the back of a desktop computer case.
10
CMOS Battery
The round silver coin cell battery on the left side. Keeps the system clock running and preserves BIOS settings even when the PC is unplugged.
11
BIOS ROM Chip
A small rectangular chip (PM25L0080) soldered directly onto the motherboard — on this board it is located near the top-right area, often marked with a sticker or the manufacturer label. This is the computer's ROM (Read-Only Memory).
12
IDE / Floppy Connector (Red)
The red ribbon cable connector (visible on left edge). A legacy interface for older storage devices. Mostly obsolete today, replaced by SATA.
13
Front Panel Headers
Small pin rows at the top of the board (labelled PANEL, F_AUDIO, USB 2.0, etc.). The case's power button, reset button, LEDs, and front USB/audio ports connect here.
🔑 Key takeaway: The motherboard is the backbone that physically and electrically connects every component. Without it, individual parts like the CPU and RAM cannot communicate. Its quality and chipset determine which CPUs, RAM types, and expansion cards are compatible.
📱
Smartphone Perspective — The Motherboard
Smartphones do not have a traditional motherboard with separate plug-in components. Instead, they use a tiny PCB (Printed Circuit Board) where almost everything — CPU, RAM, storage, modem — is permanently soldered or integrated into a single SoC (System on a Chip). This is why you cannot upgrade the RAM in your phone the way you can add a RAM stick to a desktop — it is all fused together. The trade-off is extreme compactness and power efficiency.
Going Deeper

Other Notable Components on the Motherboard

Beyond the major components listed above, a motherboard is populated with dozens of smaller specialised chips and passive components, each performing a specific job. You would find components such as: capacitors (store and smooth electrical charge, the cylindrical gold/black components), resistors (limit current flow, tiny rectangular components), MOSFETs / transistors (act as electronic switches in power circuits), operational amplifiers / op-amps (amplify and compare analog signals — e.g. the LM358D we found), crystal oscillators (generate precise clock frequencies), inductors / chokes (filter electrical noise in power lines), network / Ethernet controllers (manage wired internet), USB controllers (manage USB port communication), and clock generator chips (synchronise all components to a common timing signal). The four chips we identified on this actual board during class are highlighted below.

⚡
VRM — Voltage Regulator Module
Under the top-left heatsink
The heatsink in the top-left corner of the board covers the Voltage Regulator Module (VRM) — a circuit made of MOSFETs, inductors, and capacitors. The power supply unit (PSU) delivers 12V to the board, but the CPU needs a much lower, precisely regulated voltage (typically 0.8V–1.4V depending on load). The VRM performs this conversion in real time, thousands of times per second. This conversion process generates significant heat — which is why it needs its own heatsink.
📱 Smartphone link: Your phone's charging IC performs the same job — converting battery voltage to the precise level each chip needs. That is partly why phones warm up while charging.
🕹️
Winbond W83627DHG-P
Super I/O Controller
This is the Super I/O Controller — a single chip that manages all the legacy and low-speed input/output functions of the motherboard. It handles the keyboard and mouse ports (PS/2), serial (COM) ports, the hardware monitoring system (reading CPU temperature, fan speeds, and voltages). It acts as a bridge between the slow legacy peripherals and the faster chipset.
📱 Smartphone link: On a smartphone, similar I/O management functions are handled by small sub-chips on the PCB that manage the touchscreen controller, fingerprint sensor, and button inputs.
🔊
Realtek ALC662
HD Audio Codec · 48 pins (12 × 4 sides)
The Realtek ALC662 is the board's High-Definition Audio Codec. It is responsible for all sound on the computer. It converts digital audio signals → analog sound sent to your speakers or headphones (Digital-to-Analog Conversion, DAC), and also converts analog sound → digital when you use a microphone (Analog-to-Digital Conversion, ADC). It supports up to 5.1 surround sound output. It is located right next to the F_AUDIO header.
📱 Smartphone link: Every smartphone has an equivalent audio codec chip — when you play music through your speaker or record a voice note, this is the chip doing the digital-to-analog (and analog-to-digital) conversion. On phones it is usually integrated into the SoC.
📊
Winbond W83116WG-511
Hardware Monitor / Power Controller
The Winbond W83116WG-511 is a hardware monitoring and power management controller. It continuously monitors critical system parameters — CPU temperature, system temperature, fan rotation speeds (RPM), and supply voltages — and reports these to the Super I/O chip or directly to the BIOS. If the CPU overheats, it is this chip (working alongside the BIOS) that triggers thermal throttling (slowing down the CPU) or an emergency shutdown to prevent hardware damage.
📱 Smartphone link: Smartphones have equivalent thermal management circuits — when your phone gets too hot during gaming, the SoC automatically reduces its clock speed to cool down. This is the mobile equivalent of what this chip helps orchestrate on a desktop.
🧩 Class takeaway: A motherboard is not just a connector — it is a complete ecosystem of specialised chips, each an expert at one job. The CPU is the brain, but it depends on this entire team of support chips to communicate with the outside world, stay cool, produce sound, and receive correct voltage. Remove any one of them and a specific function stops working.
